# Русская локализация Papyrix Reader

## Что добавлено

В этом форке добавлена полная поддержка русского языка для прошивки Papyrix Reader.

### Возможности

✅ Полный перевод интерфейса на русский язык  
✅ Переключение языка через настройки  
✅ Поддержка кириллицы  
✅ Сохранение выбранного языка  
✅ Легкое добавление новых языков  

## Быстрый старт

### Выбор языка

1. Откройте **Settings** (Настройки)
2. Выберите **Device** (Устройство)
3. Найдите **Language** (Язык)
4. Выберите **Русский**
5. Интерфейс сразу переключится на русский

### Доступные языки

- **English** - английский (по умолчанию)
- **Русский** - русский

## Сборка

Проект собирается стандартным способом через PlatformIO:

```bash
# Сборка прошивки
pio run

# Прошивка устройства
pio run --target upload

# Загрузка файловой системы (если используется)
pio run --target uploadfs
```

## Добавление нового языка

Подробная инструкция в файле [`docs/localization.md`](docs/localization.md).

Краткая версия:

1. Добавить язык в `enum class Language` в `Localization.h`
2. Создать константу `STRINGS_XX` с переводами
3. Обновить метод `setLanguage()` в `Localization.cpp`
4. Добавить язык в `LANGUAGE_VALUES` в `SettingsViews.h`

## Примеры использования

### В коде C++

```cpp
#include "Localization.h"

// Использование локализованных строк
const char* text = L10N.home_no_book;
r.drawText(fontId, x, y, L10N.btn_files, color);

// Смена языка программно
papyrix::Localization::setLanguage(papyrix::Language::Russian);
```
