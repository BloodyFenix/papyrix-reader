# Сборка и тестирование локализации

## Предварительные требования

### Установка PlatformIO

#### Вариант 1: VS Code Extension (рекомендуется)

1. Установите [Visual Studio Code](https://code.visualstudio.com/)
2. Откройте VS Code
3. Перейдите в Extensions (Ctrl+Shift+X)
4. Найдите "PlatformIO IDE"
5. Нажмите Install

#### Вариант 2: PlatformIO Core CLI

```bash
# Установка через pip
pip install platformio

# Или через curl (Linux/Mac)
curl -fsSL https://raw.githubusercontent.com/platformio/platformio-core-installer/master/get-platformio.py -o get-platformio.py
python3 get-platformio.py
```

### Подключение устройства

1. Подключите Xteink X4 к компьютеру через USB
2. Убедитесь, что устройство определяется системой
3. Проверьте COM-порт (Windows) или /dev/ttyUSB* (Linux)

## Сборка проекта

### Через VS Code

1. Откройте папку `papyrix-reader` в VS Code
2. PlatformIO автоматически определит проект
3. Нажмите на иконку PlatformIO в левой панели
4. Выберите "Build" в разделе "Project Tasks"

### Через командную строку

```bash
cd papyrix-reader

# Сборка проекта
pio run

# Или с указанием окружения
pio run -e xteink-x4
```

### Ожидаемый результат

```
Processing xteink-x4 (platform: espressif32; board: esp32-c3-devkitm-1; framework: arduino)
...
Building .pio/build/xteink-x4/firmware.bin
...
========================= [SUCCESS] Took X.XX seconds =========================
```

## Прошивка устройства

### Подготовка устройства

1. Подключите Xteink X4 по USB
2. Переведите устройство в режим загрузки (boot mode):
   - Удерживайте кнопку BOOT
   - Нажмите и отпустите кнопку RESET
   - Отпустите кнопку BOOT

### Прошивка через VS Code

1. В PlatformIO sidebar выберите "Upload"
2. Дождитесь завершения прошивки

### Прошивка через командную строку

```bash
# Прошивка устройства
pio run --target upload

# С указанием порта (если не определяется автоматически)
pio run --target upload --upload-port COM3  # Windows
pio run --target upload --upload-port /dev/ttyUSB0  # Linux
```

### Ожидаемый результат

```
Configuring upload protocol...
Looking for upload port...
Auto-detected: COM3
Uploading .pio/build/xteink-x4/firmware.bin
...
Hash of data verified.
Leaving...
Hard resetting via RTS pin...
========================= [SUCCESS] Took X.XX seconds =========================
```

## Мониторинг Serial порта

### Через VS Code

1. В PlatformIO sidebar выберите "Monitor"
2. Откроется терминал с выводом Serial

### Через командную строку

```bash
# Открыть Serial Monitor
pio device monitor

# С указанием порта и скорости
pio device monitor --port COM3 --baud 115200
```

### Что смотреть в логах

```
[INFO] BOOT: Starting Papyrix Reader
[INFO] LANG: Current language: English
[INFO] HOME: Entering
[INFO] WIFI: Connecting...
```

## Тестирование локализации

### Тест 1: Проверка английского языка (по умолчанию)

1. Включите устройство
2. Проверьте главный экран:
   - Должно быть "No book open"
   - Кнопки: "Files", "Sync", "Settings"
3. Откройте Settings
4. Проверьте пункты меню: "Reader", "Device", "Cleanup", "System Info"

### Тест 2: Переключение на русский

1. Откройте **Settings**
2. Выберите **Device**
3. Найдите **Language**
4. Измените на **Русский**
5. Проверьте, что интерфейс обновился:
   - Главный экран: "Книга не открыта"
   - Кнопки: "Файлы", "Синхр.", "Настройки"

### Тест 3: Навигация по меню (русский)

1. Откройте **Настройки**
2. Проверьте пункты: "Чтение", "Устройство", "Очистка", "Информация"
3. Откройте **Чтение**
4. Проверьте настройки:
   - "Тема"
   - "Размер шрифта"
   - "Макет текста"
   - И т.д.

### Тест 4: Настройки устройства (русский)

1. Откройте **Настройки** → **Устройство**
2. Проверьте все пункты:
   - "Таймаут сна"
   - "Экран сна"
   - "При запуске"
   - "Короткое нажатие питания"
   - "Страниц до обновления"
   - "Исправление выцветания"
   - "Передние кнопки"
   - "Боковые кнопки"
   - "Язык"

### Тест 5: Переключение обратно на английский

1. Откройте **Настройки** → **Устройство** → **Язык**
2. Выберите **English**
3. Проверьте, что интерфейс вернулся на английский

### Тест 6: Сохранение настроек

1. Установите русский язык
2. Перезагрузите устройство
3. Проверьте, что язык остался русским

### Тест 7: Отображение кириллицы

1. Установите русский язык
2. Проверьте все экраны на корректное отображение:
   - Нет квадратиков вместо букв
   - Нет иероглифов
   - Все символы читаемы

### Тест 8: Длинные строки

1. Проверьте, что длинные русские строки:
   - Не обрезаются некорректно
   - Помещаются в UI элементы
   - Переносятся правильно (если применимо)

## Отладка проблем

### Проблема: Устройство не определяется

**Решение:**
```bash
# Windows: установить драйвер CH340/CP2102
# Linux: добавить пользователя в группу dialout
sudo usermod -a -G dialout $USER
# Перелогиниться

# Проверить доступные порты
pio device list
```

### Проблема: Ошибка компиляции

**Решение:**
```bash
# Очистить сборку
pio run --target clean

# Обновить платформу
pio platform update espressif32

# Пересобрать
pio run
```

### Проблема: Кириллица не отображается

**Возможные причины:**
1. Шрифт не поддерживает кириллицу
2. Неправильная кодировка файлов (должна быть UTF-8)
3. Проблема с отображением на E-Ink дисплее

**Решение:**
- Проверьте, что файлы сохранены в UTF-8
- Убедитесь, что используемые шрифты поддерживают кириллицу
- Проверьте логи на ошибки

### Проблема: Язык не сохраняется

**Решение:**
- Проверьте, что настройки записываются в файл
- Проверьте права доступа к файловой системе
- Проверьте логи на ошибки записи

### Проблема: Интерфейс не обновляется после смены языка

**Решение:**
- Убедитесь, что все UI компоненты используют `L10N` макрос
- Проверьте, что вызывается перерисовка после смены языка
- Попробуйте перезагрузить устройство

## Создание релиза

### Сборка финальной прошивки

```bash
# Сборка в release режиме
pio run -e xteink-x4

# Прошивка находится в:
# .pio/build/xteink-x4/firmware.bin
```

### Создание архива

```bash
# Создать архив с прошивкой
mkdir release
cp .pio/build/xteink-x4/firmware.bin release/
cp LOCALIZATION_RU.md release/
cp docs/localization.md release/

# Создать zip
zip -r papyrix-localized-v1.0.zip release/
```
